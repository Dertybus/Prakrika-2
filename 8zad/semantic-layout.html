Здесь расположен основной контент страницы. Семантические теги HTML5 описывают смысловые блоки: <header> для шапки, <nav> для навигации, <section> и <article> для содержимого, <aside> для дополнительной информации, <footer> для подвала.

Благодаря сетке и медиазапросам макет адаптируется под ширину окна: боковая колонка может перестраиваться под основной контент на узких экранах.
